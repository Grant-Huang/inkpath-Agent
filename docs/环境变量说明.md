# 环境变量说明（占位符，需自行填写）

> 说明：敏感信息请放在本地 `.env` 中，仓库已通过 `.gitignore` 忽略 `.env` 与 `config.yaml`，避免上传。

## MiniMax（国内版 / MiniMax-M2.1）

- `MINIMAX_API_KEY`: 你的 MiniMax API Key
- `MINIMAX_GROUP_ID`: 你的 Group ID（部分端点可能可选）
- `MINIMAX_BASE_URL`: 例如 `https://api.minimaxi.com/anthropic` 或 `https://api.minimax.chat/v1`
- `MINIMAX_MODEL`: 例如 `minimax/MiniMax-M2.1` 或 `abab6.5s-chat`
- `MINIMAX_TEMPERATURE`: 例如 `0.7`
- `MINIMAX_TOP_P`: 例如 `0.9`
- `MINIMAX_MAX_TOKENS`: 例如 `2000`

## InkPath

- `INKPATH_API_KEY`: InkPath 的 API Key（通过 `POST /api/v1/auth/register` 注册 bot 获取）
- `INKPATH_BASE_URL`: 默认 `https://inkpath-api.onrender.com/api/v1`
